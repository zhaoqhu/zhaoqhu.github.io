Mac VS Code 集成 claude-code 教程

发表于 2026-05-06 | 更新于 | 分类于 AICoding

介绍

claude-code 是一款强大的终端智能开发工具，能在 VS Code 中为我们提供实时的代码辅助、问题解答和逻辑优化能力。但官方默认对接 Anthropic 的 Claude 模型，存在网络访问、账号注册、付费等诸多门槛。本文将手把手教你完成 claude-code 的安装、VS Code 插件配置，并重点讲解如何通过 cc-switch 对接国内 DeepSeek 大模型，享受更便宜、更快速的本地化智能开发体验。

环境准备

安装VS Code

官网下载并安装VS Code，后续在VS Code中安装claude-code插件。

安装cc-switch

一个用于管理和切换Claude后端配置的工具，mac选择.dmg安装。

  • 下载地址 https://github.com/farion1231/cc-switch/releases/tag/v3.14.1
  • 选择 https://github.com/farion1231/cc-switch/releases/download/v3.14.1/CC-Switch-v3.14.1-macOS.dmg

安装Node.js

运行Claude-code 需要Node.js环境，请确保你的系统已经安装了 Node.js。

安装nvm

1
2
brew install nvm

配置nvm环境变量

把nvm配置添加到zshrc文件中，并执行sourcezshrc命令使配置生效。

1
2
3
4
5
6
7
grep -qxF '# NVM Configuration' ~/.zshrc || cat << 'EOF' >> ~/.zshrc

# NVM Configuration
export NVM_DIR="$HOME/.nvm"
[ -s "/usr/local/opt/nvm/nvm.sh" ] && . "/usr/local/opt/nvm/nvm.sh"
[ -s "/usr/local/opt/nvm/etc/bash_completion.d/nvm" ] && . "/usr/local/opt/nvm/etc/bash_completion.d/nvm"
EOF

重新加载zshrc配置

1
source ~/.zshrc

安装Node.js

安装指定版本的Node.js

1
nvm install 23.2.0

验证Node.js版本

1
node -v

安装claude-code

执行安装命令

1
npm install -g @anthropic-ai/claude-code

查看安装位置

1
2
$ which claude
/Users/zhaoqinghu/.nvm/versions/node/v23.2.0/bin/claude

claude创建软连接

claude-code 默认安装在nvm的node目录下，为了方便使用，我们可以创建一个软链接到/usr/local/bin目录。

1
ln -s /Users/zhaoqinghu/.nvm/versions/node/v23.2.0/bin/claude /usr/local/bin/claude

在VS Code中安装claude-code插件

  1. 打开VS Code，点击扩展图标，搜索并安装Claude Code for VS Code插件。
  2. 重启VS Code，确保插件正常加载。

配置cc-switch，对接DeepSeek大模型

登录DeepSeek账号，获取API Key

https://platform.deepseek.com/api_keys

打开cc-switch，配置DeepSeek API

添加，选择DeepSeek供应商模型，输入API Key。
