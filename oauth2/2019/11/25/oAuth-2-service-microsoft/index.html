function get_url($url,$data){
	$ch = curl_init();
	curl_setopt($ch, CURLOPT_URL, $url);
	curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
	curl_setopt($ch, CURLOPT_SSL_VERIFYPEER, FALSE); 
	curl_setopt($ch, CURLOPT_SSL_VERIFYHOST, FALSE); 
	curl_setopt($ch, CURLOPT_CUSTOMREQUEST,"GET"); 
	curl_setopt($ch, CURLOPT_HTTPHEADER, $data);
	curl_setopt($ch, CURLOPT_PROXY, '127.0.0.1');
	curl_setopt($ch, CURLOPT_PROXYPORT, '1080');
	$output = curl_exec($ch);
	if($output === false)
	  {
	      echo 'Curl error: ' . curl_error($ch);
	  }
	curl_close($ch);
	return $output;
}

App registrations创建步骤

1、访问https://portal.azure.com/#allservices
2、Identity=>App registrations=>New registration
3、registration主要参数有Application (client) ID，client secrect,创建方法选中已创建的registration=>Certificates & secrets=> New client secrect
4、开发过程中程序代码里client_id ，client_secret，redirect_uri，这三项的值，需要registration提供。
